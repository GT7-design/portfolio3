# 서울풍납초등학교 스마트 보결 배정 시스템 (포트폴리오 3)

본 프로젝트는 초등학교 내 결강 및 보강 상황 발생 시, 보결 가능 교사를 추천하고 수업 안내문을 자동으로 작성하며, 교사 간 실시간 소통을 지원하는 스마트 배정 시스템입니다.

## 🚀 주요 기능
- **구글 계정 연동 로그인**: Firebase Authentication을 활용하여 보안이 강화된 교직원 인증 (Google 계정)
- **보결 추천 시스템**: 해당 교시와 교과에 맞춰 보결 수업이 가능한 교사를 자동으로 필터링 및 추천
- **실시간 교사 채팅**: Firebase Realtime Database 기반의 실시간 채팅으로 교직원 간 빠른 소통 지원
- **안내문 자동 완성 및 공유**: 보결 요청 시 입력한 정보를 바탕으로 수업 안내문을 자동 생성하고, 원클릭으로 채팅창에 공유 가능

## 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend & Database**: Firebase Authentication (Google 소셜 로그인), Firebase Realtime Database
- **Deployment**: Vercel (또는 Firebase Hosting)

## 📝 사용 방법
1. **로그인**: `Google 계정으로 인증` 버튼을 클릭하여 로그인합니다.
2. **보결 요청서 작성**: 결강 교사, 교시, 학급, 교과 등의 정보와 할 일 및 유의사항을 입력합니다.
3. **추천 및 안내문**: `보결 가능 교사 추천` 버튼을 누르면 추천 결과가 표시되고 안내문이 자동 생성됩니다.
4. **채팅 공유**: 생성된 안내문을 복사하거나 `채팅창에 공유` 버튼을 통해 실시간 채팅방에 전달할 수 있습니다.
